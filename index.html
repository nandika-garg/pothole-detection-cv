Pothole Detection using Computer Vision

Automatic detection of potholes using CNN for safer roads.

Problem Statement

Road potholes cause accidents and damage vehicles. Manual inspection is slow and inefficient. This project uses computer vision to automatically detect potholes from images.

Literature Review

Existing systems use sensors and manual inspection, which are costly.

Research Gap

Need for low-cost, automated pothole detection using deep learning.

Methodology

Dataset

Images of potholes and plain roads collected from internet. Images are resized and converted to grayscale.

Model

CNN model with Conv2D layers, ReLU activation, pooling, and dense layers.

Demo

Results

Accuracy

~60%

Model successfully detects potholes from input images.
